# data/seeds

Esta carpeta contiene **archivos de semillas** para la clasificación del Marco 2025.

Las **semillas** son listas de indicadores, términos y definiciones asociadas a:

- MARCO_CATEGORIA
- MARCO_SUBCATEGORIA
- Sector Elegible

Se utilizan en el módulo de NLP para asignar automáticamente categoría a los proyectos
a partir de sus brechas.

Archivos esperados:
- `Semillas_Indicadores_Actualizados.xlsx`

Este archivo NO se versiona automáticamente; debe colocarse manualmente en esta carpeta.
